After finishing my MSc. at the BLINC Lab, I stuck around for a few months to add a generic pattern recognition tool to their open-source prosthesis control infrastructure, brachI/Oplexus. I added a generic interface to facilitate data collection, model design, model training, and output mapping of the model predictions to prosthesis motion. You can find the code on my github.
